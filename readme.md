# MayDOS Community Edition [mdsc]
本版本重写了MayDOS大部分函数，对部分功能进行了删减，后续会进行整合  

-使用手册：
--           echo        在屏幕上打印字符串  
--            sof           执行任意可执行文件  
--           sysver      显示系统版本  
--           exit          退出MayDOS  
--           del           删除文件  
--            cat           读取文件并显示  
--            mplay      播放音乐  
--            clear         清屏  
--            cuser        创建新用户  
--            password 修改用户的密码  
--            logout      登出MayDOS  
--            直接输入将会执行bin和app目录下的文件
